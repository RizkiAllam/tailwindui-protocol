# Business API

Selamat datang di dokumentasi Business API WhatsApp. API ini memungkinkan bisnis untuk terhubung dengan pelanggan mereka melalui WhatsApp, memberikan pengalaman interaksi yang kaya dan personal.

## Business Model

Model bisnis mencakup semua informasi yang terkait dengan bisnis yang menggunakan API WhatsApp. Bisnis dapat mengelola profil, mengirim pesan, dan berinteraksi dengan pelanggan mereka.

### Properties

<Properties>
  <Property name="id" type="string">
    Pengidentifikasi unik untuk setiap bisnis.
  </Property>
  <Property name="name" type="string">
    Nama bisnis.
  </Property>
  <Property name="industry" type="string">
    Industri bisnis.
  </Property>
  <Property name="location" type="string">
    Lokasi fisik bisnis.
  </Property>
  <Property name="contact" type="string">
    Informasi kontak bisnis.
  </Property>
  <Property name="profile_image" type="string">
    URL gambar profil bisnis.
  </Property>
  <Property name="is_verified" type="boolean">
    Apakah bisnis ini telah diverifikasi atau tidak.
  </Property>
</Properties>
